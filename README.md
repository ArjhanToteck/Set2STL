# Set2STL
This project uses the [Rebrickable API](https://rebrickable.com/api/) and my [LDraw Library repository](https://github.com/ArjhanToteck/LDraw-Library) to convert a Lego set ID into an STL file. It is intended solely for personal, educational, or non-commercial use.
